# Contributing

- Pull requests are welcome.
- For major changes or new features, open an issue first to discuss the change. Small changes don't need one.
- Build and test from the repo before working:

```sh
git clone https://github.com/emiliolugo/clawmark.git
cd clawmark
cargo build
```

- After changes, run:

```sh
cargo test
```

- Update tests as appropriate.
